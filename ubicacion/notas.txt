Método HTTP	URL	Acción
GET	/ubicaciones/	Lista todas las ubicaciones
POST	/ubicaciones/	Crear una nueva ubicación
GET	/ubicaciones/{id}/	Detalle de una ubicación
PUT	/ubicaciones/{id}/	Actualizar completamente
PATCH	/ubicaciones/{id}/	Actualizar parcialmente
DELETE	/ubicaciones/{id}/	Eliminar una ubicación


si se utiliza los filtros pueden buscar direcciones asi:

/ubicaciones/?departamento=Santander

/ubicaciones/?ciudad=Bucaramanga

/ubicaciones/?tipo_ubicacion=barrio

ejemplo con filtros combinados:

/ubicaciones/?departamento=Santander&ciudad=Bucaramanga&tipo_ubicacion=barrio
